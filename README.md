# Portfolio 05 - React Hooks App
Simple React Hooks App using Vite + React + Typescript

# Topics
- useState
- useEffect
- useRef
- useLayoutState
- useMemo
- useCallback
- useReducer
- Custom hooks
    - useFetch
    - useCounter
    - useForm
    - useTodo


# Installation
```
cd hooks-app
yarn install
yarn dev
```

# Build
```
yarn build
```
